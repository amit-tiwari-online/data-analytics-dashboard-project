# Data Analytics Dashboard Project

## 📊 Overview
This project focuses on analyzing software performance and building a dashboard for monitoring system activity.

## 🛠️ Tools Used
- Python (Pandas, Matplotlib)
- Power BI
- Excel

## 📈 Features
- Success vs Failure Analysis
- Processing Time Trends
- User Activity Tracking
- KPI Dashboard

## 📌 Insights
- High failure rate for large files
- Peak usage at specific times
- Performance optimization needed
